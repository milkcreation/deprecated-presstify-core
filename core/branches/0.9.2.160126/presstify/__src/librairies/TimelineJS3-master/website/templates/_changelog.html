TimelineJS is Knight Lab's most widely used product. TimelineJS 3 is a full rewrite designed to make this popular tool more flexible for authors and developers.

Screenshot of TimelineJS 3. Click to see timeline.

TimelineJS 3 is still under development. Current users can see how existing timelines would look. (If you're brand new, stick with the original TimelineJS for now.)

If you want to help us out, make a few timelines and let us know if anything goes wrong. More technical users can help us find and fix obscure bugs.

What's changed?

  • Flexible row layout: no longer limited to 3/6 rows.
  • Encapsulation: More than one timeline per page.
  • Support for very large time scales: dinosaurs! the Big Bang!
  • New media types: Dropbox images; DocumentCloud
  • Dropped support for Pagepeeker (screenshots of websites)

What's coming?

  • Spreadsheet-based data format
  • Support for "tags"
  • Support for "eras"
  • Handling fonts and other custom presentation
